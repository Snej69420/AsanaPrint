# Welcome to Asana Gantt Viewer

**Asana Gantt Viewer** is a desktop application that transforms standard Asana CSV exports into interactive, high-quality Gantt charts.

![App Screenshot](assets/example.png){ align=right width=300 }

It is designed to bridge the gap between simple task lists and complex project management visualization, allowing you to filter, style, and export your project timelines without needing a paid Asana premium subscription for Timeline view.
